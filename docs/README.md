| File | Audience | Purpose |
|------|----------|---------|
| `how_to_reproduce.md` | Researchers | Step-by-step guide to reproduce the BQAI analysis from clean environment. |
| `faq.md` | Reviewers, readers | Anticipated questions about methodology, sample selection, IAA, etc. |

| `BQAI_scoring_template_ES.xlsx` | Evaluators (Spanish) | The Excel template used by evaluators to record their scores (30 benchmarks × 7 dimensions). |

## Languages

The evaluator-facing materials are in **Spanish** because the evaluators (the paper's co-authors and collaborators) work primarily in Spanish. The repository documentation, code, and scientific output are in **English** for international audience.

If you want to run a re-scoring exercise with English-speaking evaluators, contact us — we have an English version of the protocol available.
